# Sample media

Put your test traffic **images** (`.jpg`, `.png`) and **videos** (`.mp4`, `.avi`,
`.mov`) here, then reference them from the CLI or upload them in the dashboard.

Good free sources of traffic footage for testing:
- Pexels / Pixabay (search "traffic road cars") — free stock videos.
- Any dashcam / CCTV clip of a road.

Example:
```bash
python scripts/detect_cli.py --image data/samples/road.jpg --location "KN 1 Rd - City Centre"
```
